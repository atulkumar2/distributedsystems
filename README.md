# Vehicle Telemetry Kafka — Learning Project

Two self-contained setups for learning Java + Apache Kafka integration using an IoT/vehicle-telematics scenario.

| Setup | What it is |
|---|---|
| [basic-cli-setup/](basic-cli-setup/) | Plain Java CLI producer and consumer — minimal, no frameworks, great starting point |
| [web-app-setup/](web-app-setup/) | Spring Boot web apps — generate events from a browser form, watch them appear live in another tab |

## Prerequisites

- Docker & Docker Compose (both setups use Compose to start Kafka)
- Java 25+
